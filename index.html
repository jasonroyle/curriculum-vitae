Experience

Most of my projects have been created using CakePHP, an object-orientated and MVC based PHP framework. I have started to develop new projects in the Lithium framework, which is similar to CakePHP but built upon PHP version 5.4 or greater. Together with Lithium I use a no-SQL, document based database system called MongoDB. All my projects are under git version control.

I currently have two VPS servers that I have setup and configured myself, which both run CentOS and have iptables, postfix, httpd, php, mysql and mongodb services running. My responsibilities in employment have ranged from creating secure systems to fulfilling my role as 'Technical Support Manager' at conference events where I set up the registration area with a Linux server and laptops and assist speakers with their presentations.
